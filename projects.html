Puzzle Game and Solver (Python)

This application is a replication of a game called Nonogram or Picross. It is written in python using the library pygame. In the game there is a main menu to select the level. In the level there are three lives and selecting the wrong box loses a life. After all lives are lost the level is over.

Online Grocery Store

This project was from my class Database Organization. The application uses Java and JavaFX for the user interface. In the application there are different views for both employees and customers. Customers can change information, add credit cards, chack balances, add items to their cart, and finally place an order. The picture to the right displays the Item Search view where customers can look for items to buy and then add them to their cart.

Website Stock Scraper

This project is a python script for the purpose of getting information about certain types of activity around stocks. The script is automated by a scheduled task running multiple times a day to update myself on stocks. It scrapes info from Finviz.com to get the stocks and then it retrieves real time data from Yahoo Finance. The stocks come from the following categories.

  • Top Gainers
  • Over Bought
  • Unusual Volume
  • Most Volatile
